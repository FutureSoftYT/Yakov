1. Изменить данные в config.py  ../data/config.py
Там есть такие поля как LOGIN_INS,PASSWORD_INS туда записиваются логин и парол инстаграма

И там ж есть поля админс туда записивается ИД Телеграм как пример [12345, 3212312,123123,..]

2. Установка библиотек pip install -r requirements.txt Убедитесь что вы находитесь в директории TelegramBot)

3. После установки запустим 2 скрипта

python app.py #Запустится бот 
python adminpanelrun.py runserver 0.0.0.0:8000   #Запуск веб админки. Доступ админки будет по этой ссылки 190.2.140.239:8000 Логин : Yakov  Пароль: Yakov2022
